# OKX Autonomous Bot

Deploy-ready Python bot for automated trading on OKX.

## Setup

1. Upload to GitHub
2. Connect to Railway
3. Set environment variables:
   - OKX_API_KEY
   - OKX_SECRET_KEY
   - OKX_PASSPHRASE
   - DRY_RUN (true/false)

## Usage

Bot runs main.py and evaluates signals every 60s.
